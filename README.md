# 🧠 AI Surveillance System

A real-time AI-based CCTV web app that performs **face detection, emotion analysis, and basic threat monitoring** directly in the browser.

---

## 🚀 Features

* 🎥 Live camera feed
* 👤 Multi-face detection
* 😊 Emotion recognition (happy, sad, angry, neutral)
* 🚨 Basic threat detection (motion, no-face, camera block)
* 🟢🔴 Admin / Intruder labeling
* 🖥️ Clean dashboard UI

---

## 🛠️ Tech Stack

* React (Vite + TypeScript)
* TailwindCSS
* face-api.js

---

## ⚙️ Run Locally

```bash
git clone https://github.com/your-username/ai-surveillance-app.git
cd ai-surveillance-app
npm install
npm run dev
```

Open: http://localhost:5173
